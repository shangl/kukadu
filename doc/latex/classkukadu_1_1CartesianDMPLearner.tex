\subsection{Detailed Description}
The Trajectory\+D\+M\+P\+Learner encapsulates the dmp learning process. 

Dynamic movement primitives can be easily learned by using this class and providing the cartesian data or a file containing this data. Basically this is a helper that enables the programmer to reduce code complexity.
